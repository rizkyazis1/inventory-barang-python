# Inventory Management System

Final Project Mata Kuliah Struktur Data.

## Deskripsi
Aplikasi manajemen inventori barang berbasis Python yang menggunakan database flat file (.CSV). Aplikasi ini mendukung operasi CRUD (Create, Read, Update, Delete) serta mengimplementasikan struktur data Linked List dan Queue, algoritma Bubble Sort, dan Linear Search.

## Fitur
- Tambah Barang
- Tampilkan Barang
- Cari Barang
- Edit Barang
- Hapus Barang
- Sorting Barang (Bubble Sort)
- Riwayat Barang Masuk (Queue)

## Struktur Data
- Linked List
- Queue

## Algoritma
- Bubble Sort
- Linear Search

## Database
- CSV (Flat File)

## Bahasa Pemrograman
- Python
